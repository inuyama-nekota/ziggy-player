# ZIGGY PLAYERをiPhoneへ入れる手順

最初の1回だけ、PCでGitHubへプレイヤー本体を置きます。その後はiPhoneのホーム画面から普通のアプリのように起動できます。

## 先に確認するもの

PC側に次の4ファイルを用意してください。

- `01 STAY GOLD.m4a`
- `STAY GOLD.lrc`
- `04 GLORIA.m4a`
- `gloria.lrc`

この4ファイルはGitHubへ載せません。AirDrop、iCloud Drive、メール添付など、使いやすい方法でiPhoneの「ファイル」アプリへ保存しておきます。

## 1. GitHubへプレイヤー本体を置く

1. PCのブラウザーで [GitHub](https://github.com/) を開いてログインします。
2. 右上の「＋」から **New repository** を選びます。
3. Repository nameへ `ziggy-player` と入力します。
4. **Public** を選び、**Create repository** を押します。
5. 作成された画面で **uploading an existing file** を選びます。
6. このプロジェクトフォルダーの中身をすべてドラッグします。
7. ファイル一覧にM4AやLRCがないことを確認します。
8. 下部の **Commit changes** を押します。

アップロードするのは `index.html`、`app.js`、`styles.css`、`icons` フォルダーなどのプレイヤー本体だけです。

## 2. URLを発行する

1. GitHub上のリポジトリで **Settings** を開きます。
2. 左側の **Pages** を選びます。
3. Build and deployment の Sourceで **Deploy from a branch** を選びます。
4. Branchを **main**、フォルダーを **/(root)** にして **Save** を押します。
5. 数分待って画面を更新します。
6. `Your site is live at ...` と表示されたURLを開きます。

URLは通常、次のような形になります。

```text
https://あなたのGitHub名.github.io/ziggy-player/
```

## 3. iPhoneのホーム画面へ追加する

1. 発行されたURLをiPhoneの **Safari** で開きます。
2. 画面下部の共有ボタン（四角から上向き矢印）を押します。
3. **ホーム画面に追加** を選びます。
4. 「Webアプリとして開く」が表示された場合はオンにします。
5. 右上の **追加** を押します。

ホーム画面に金色のZアイコンが追加されます。

## 4. 2曲を登録する

ホーム画面のZIGGY PLAYERを起動すると、最初に登録画面が出ます。

1. STAY GOLDの「音源を選ぶ」から `01 STAY GOLD.m4a` を選びます。
2. STAY GOLDの「歌詞を選ぶ」から `STAY GOLD.lrc` を選びます。
3. GLORIAの「音源を選ぶ」から `04 GLORIA.m4a` を選びます。
4. GLORIAの「歌詞を選ぶ」から `gloria.lrc` を選びます。
5. **このiPhoneに保存** を押します。

「保存しました」と表示されたら準備完了です。

## 普段の操作

- 曲名部分を上下へスワイプ：曲を選ぶ
- 中央の金色ボタン：再生／一時停止
- 左右のボタン：前曲／次曲
- 横長のバー：再生位置を移動
- 右上の歯車：音源や歌詞を選び直す

歌詞は「前の行・現在行・次の行」の3行だけ表示されます。現在行は常に中央で太字になります。

## 困ったとき

### URLは開くがホーム画面へ追加できない

LINEやメール内のブラウザーではなく、Safariで直接URLを開いてください。

### 音が出ない

iPhoneの消音設定と音量を確認し、中央の再生ボタンを一度押してください。iPhoneでは安全上、自動で最初の音を鳴らせません。

### 歌詞が動かない

歯車を押し、M4AとLRCの組み合わせが正しいか確認してください。LRCには `[00:12.34]` のような時刻が必要です。

### 登録した曲が消えた

Safariの履歴・Webサイトデータを消した場合や、iPhoneが保存領域を整理した場合は再登録が必要です。元の4ファイルは「ファイル」アプリへ残してください。

### アプリを更新したい

GitHub上のプレイヤーファイルを入れ替え、ホーム画面のアプリをいったん終了して再度開きます。登録した音源と歌詞は通常そのまま残ります。
